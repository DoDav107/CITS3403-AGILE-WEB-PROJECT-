
# BiteScout Frontend Starter

This is a front-end starter for your Agile Web Development food discovery project.

## Included pages
- Home page
- Sign up page
- Login page
- Browse/search restaurant page
- Restaurant details page
- Dish details page
- Write review page
- Edit review page
- User profile page
- Other users profile page
- Recommendations page
- Favourites page
- About/help page
- Logout page

## Tech
- HTML
- CSS
- JavaScript
- Bootstrap
- Browser localStorage
- Browser Geolocation API

## How to run
Open `index.html` in a browser.

For the geolocation feature, run the site with a local server or localhost for the best chance of browser permission working correctly.

## Notes
This is a front-end prototype.
- Login and signup are mocked with localStorage.
- Reviews and favourites are stored in localStorage.
- Real security, account handling, and persistent multi-user data should be implemented later with your Flask backend.
