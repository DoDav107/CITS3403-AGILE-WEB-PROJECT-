User Stories

. As a new user, I would want to create an account, so that I can save my reviews and preferences

. As a registered user, I would like to log in and log out securely, so that my account and activity are protected

. As a user, I want my profile information, reviews and ratings to be saved between sessions so that my data is not lost when I leave the site

. As a user, I want recommendations of a list of places to discover and eat around my location

. As a user, I want to search for places to eat/drink by name, location etc so that I can quickly find places that match what I want

. As a user, I want to filter retreatants by category such as cuisine, price range or rating, so that I can narrow down my options

. As a user, I want to rate restaurants and individual dishes etc so that I can share my experience with others

. As a user, I want to be able to write, edit and delete my own reviews so that I can manage what I post on the website

. As a user, I want to see reviews and ratings from others so that I can make better decision based on community feedback

. As a user, I want to be able to save favourite restaurants or dishes, so I can return to them later

List of main MVP pages website should have

. Home page

. Sign up page

. Login page

. Browse/Search restaurants page

. Restaurant details page

. Write/Edit review page

. User profile page

These pages cover the core MVP flow: account creation, login, discovering restaurants, viewing restaurant information and reviews, posting reviews, and managing a user's own activity. Features such as dish-specific pages, recommendations, favourites, other user profiles, and help pages can be added in later versions.

Bootstrap

Proposed Application

Our proposed application is a food discovery and review website where users can create an account, log in, browse restaurants and dishes, leave ratings and reviews and receive simple recommendations based on preferences and community feedback.

The main aim of this website is to help users discover places to eat more easily while also building a community where users can share their opinion and experience with food.
