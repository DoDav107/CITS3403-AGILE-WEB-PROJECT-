About the project

About & Help

What the app does and how discovery, reviews, and favourites work.

Application aims

BiteScout helps people discover restaurants and cafes, save favourites, and share reviews in one simple food discovery app.

User journey

  • Create an account or log in.
  • Browse restaurants near a chosen area or current location.
  • Filter places by cuisine, place type, and star rating.
  • Open restaurant and cafe pages for more detail.
  • Write reviews, save favourites, and receive recommendations based on preferred cuisine.

Help

For questions or support, email the BiteScout team at bitescout.team@gmail.com.

Geolocation usually works on localhost or HTTPS. Some browsers block location access from plain file paths.

Recommendations start with your preferred cuisine, then include a smaller mix of other cuisines so the results still feel varied.

Restaurant data, ratings, and recommendations are for discovery only. Users should check current opening hours, menus, prices, and dietary information with the venue before visiting.

Roadmap

The roadmap is the planned next set of improvements for BiteScout.

  • Allow users to request missing restaurants or places that are not found in the app.
  • Improve recommendations using saved favourites, review history, and preferred cuisine.
  • Add stronger profile customisation with avatars, uploaded photos, and editable bios.
  • Add moderation tools for inappropriate reviews and incorrect venue information.

Missing place requests

If a restaurant is not available, users can send a request with the place name, suburb, cuisine type, and any useful notes.
