About & Help

What the app does, how to use it, and which features are currently mocked on the frontend.

What BiteScout does

  • Create accounts and log in on the frontend demo
  • Search and filter restaurants by cuisine, price, rating and suburb
  • Open restaurant and dish details pages
  • Write, edit and delete reviews
  • Save favourites
  • Use browser location to find nearby spots

Help

Geolocation usually works on localhost or HTTPS. Some browsers block it on plain file paths.

This starter version uses localStorage for a front-end demo. Real security should be built later with your Flask backend.

Nearby recommendations are sorted using geographic distance and then displayed with rating and cuisine information.
