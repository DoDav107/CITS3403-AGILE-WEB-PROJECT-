About & Help

What the app does and how discovery, reviews, and favourites work.

What BiteScout does

  • Create accounts and log in with the Flask backend
  • Browse nearby Google Places restaurants using your current location
  • Search and filter places by type, tag, and star level
  • Open restaurant and dish details pages
  • Write, edit and delete reviews
  • Save favourites
  • Get recommendations from your favourite cuisine

Help

Geolocation usually works on localhost or HTTPS. Some browsers block it on plain file paths.

Log in, open a restaurant or dish, then save it from the quick actions panel.

Recommendations start with your favourite cuisine, then add a smaller set of other cuisines so the list still has variety.
