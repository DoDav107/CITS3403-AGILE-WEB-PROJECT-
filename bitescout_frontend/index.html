Location feature

Use current location to find food and drinks nearby

The recommendations page can request the user’s current location through the browser and sort sample restaurants by nearest distance. This is a strong feature to show in your Agile project demo.

Demo flow

  1. Click “Use my location”
  2. Allow browser location access
  3. See nearby places ranked by distance + rating
  4. Open a restaurant and save it to favourites
